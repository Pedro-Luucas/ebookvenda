  Sobre o ebook

  Título: Claude Code na Prática: Crie Seu Sistema de Agendamento do Zero

  cores: cores do claude code

  Objetivo declarado no ebook: Este é um tutorial completo e linear para criar, do absoluto zero, um sistema de agendamento para uso pessoal e local. O sistema é feito para rodar no próprio       
  computador do dono do negócio, sem precisar de internet, servidor externo ou infraestrutura complexa. Serve para barbearias, consultórios, estúdios, ou qualquer pessoa que precise organizar     
  horários. Não há autenticação de usuários: o sistema é de uso exclusivo do dono, que gerencia tudo. Este ebook cobre apenas a versão inicial do sistema. Evoluções e novas funcionalidades ficam  
  fora do escopo.

  Público-alvo: Pessoas com conhecimento mínimo do Claude Code que nunca programaram e querem criar um software real para uso próprio.

  Tom: Direto, acessível, sem jargão técnico desnecessário. Quando um termo técnico for inevitável, explique-o imediatamente em linguagem simples. Sem ser motivacional. Apenas claro e útil.

  Formato: Ebook completo com texto corrido, capítulos bem desenvolvidos, subtítulos, listas onde couber e blocos de código para comandos reais.

  Stack técnica do sistema a ser construído

  - Frontend: interface web simples, visual limpo e funcional (HTML, CSS, JavaScript básico ou framework leve)
  - Backend: Node.js com Express (ou equivalente simples)
  - Banco de dados: SQLite, rodando localmente no computador
  - Deploy: local, sem Docker, sem servidor externo. O sistema roda no computador do dono e é acessado pelo navegador via localhost

  O Claude Code vai configurar tudo isso. O leitor não precisa entender o que é cada tecnologia em profundidade, apenas seguir o tutorial.
